# 🚀 Meu Portfólio Pessoal

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para centralizar meus projetos, habilidades e experiências na área de Desenvolvimento Web, com foco em uma interface moderna, responsiva e de alta performance.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

* **React.js** - Biblioteca principal para construção da interface.
* **Vite** - Build tool de última geração para um desenvolvimento rápido.
* **Tailwind CSS** - Framework CSS utilitário para estilização rápida e responsiva.
* **Framer Motion** - Biblioteca para animações fluidas e interações modernas.
* **JavaScript (ES6+)** - Linguagem base do projeto.

## ✨ Funcionalidades

* **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.
* **Modais de Vídeo:** Visualização dinâmica dos projetos em execução.
* **Navegação Fluida:** Menu hamburger para dispositivos móveis e âncoras suaves.
* **Estética Moderna:** Uso de técnicas como Glassmorphism e Dark Mode.
* **Seção de Habilidades:** Listagem clara de tecnologias dominadas e em estudo.
